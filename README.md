# caixaeletronicodigital
JS + Python / Plataforma de caixa eletrônico digital criada para fins de aprendizado. Cada usuário possui login, senha e saldo próprio, podendo realizar operações de depósito e saque. O projeto é desenvolvido sem uso de frameworks , priorizando o entendimento da lógica, estruturas de dados e comunicação entre front e back-end.
